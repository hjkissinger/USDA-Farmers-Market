# USDA Farmers Market Registrations Post-Pandemic: Transitioning to Online Directories (2023-03-14) :apple: :corn: :rooster:

This was a data cleaning and analysis project examining yearly USDA farmers market registrations in an online directory from 2020-2022. The tools used for this project were R, Google Sheets, and Tableau. Check out the <a href=>R Scripts</a> or head to the <a href="https://public.tableau.com/app/profile/hannah.kissinger6750/viz/FarmersMarketsintheU_S_/USDAFarmersMarketRegistrationsPost-Pandemic">Tableau</a> visualizations to see the interactive dashboard!
